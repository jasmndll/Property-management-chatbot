# Property Management Chatbot

A Python-based chatbot designed to assist with property management-related queries such as property details, tenant support, and general inquiries. The project demonstrates how conversational agents can be applied to automate communication and improve efficiency in property management systems.

## Overview

The Property Management Chatbot is a conversational application that interacts with users and provides automated responses for common property-related questions. It reduces manual effort by handling routine inquiries and serves as a foundational implementation for intelligent property management systems.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/jasmndll/Property-management-chatbot.git
2. Navigate to the project directory:
   cd Property-management-chatbot
3. Create and activate a virtual environment:
   python -m venv venv
   source venv/bin/activate   # Linux / macOS
   venv\Scripts\activate      # Windows
4. Install dependencies:
   pip install -r requirements.txt
5. Run the chatbot using:
   python chatbot.py
